藥品外觀 · 重要

關於藥丸的誠實話

demo 藥的外觀目前是「示意」，還沒接食藥署資料

你特別要求藥丸要像真的——我同意，這是醫療 App 的底線。但要做到每顆都對，得串接食藥署「藥品外觀資料集」。我查證後發現該開放資料 API 已改版、目前抓不到資料，所以我沒有虛構任何「真實外觀」假裝核實過。

取而代之，我做了一份稽核報告：逐顆把「App 引擎畫的」和「真實藥型該長的樣子」並排，誠實標出引擎現在做得到（圓／橢圓／膠囊）、做不到（菱形／三角／印字／十字刻痕）的地方，並寫好上線前要接的完整資料管線。
