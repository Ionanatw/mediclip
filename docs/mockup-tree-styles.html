櫻花樹 — 三種風格試畫

同一棵「茁壯期櫻花樹」，用你給的三張參考圖風格各畫一次。選一個，整個快樂花園（5 個成長階段 × 3 個樹種）都會用這個風格延伸。

最貼近 Phantom

A描邊錯位塗鴉風

粗深紫輪廓＋色塊故意錯位（粉色往右上偏移 10px）、輪廓白花、檸檬花蕊、薰衣草紫星芒塗鴉。手繪感、活潑、有個性 — 和 Phantom 官網插畫同語言，UI 上的深紫描邊也能呼應。

推薦 — 與 Phantom UI 元件最一致

B霧面光暈漸層風

無輪廓、柔焦徑向光暈。黃昏紫粉天空、樹冠像發光的雲、深紫剪影枝幹。最安靜、最「呼吸練習」的氛圍，但和 Phantom 的粗線活潑感距離較遠，成長階段差異也較難畫清楚。

適合呼吸練習頁背景，當主樹偏靜

C有機色塊拼貼風

不規則色塊半透明疊加＋紋理。三層暮粉色塊堆出樹冠，點點與斜線紋理、芥末黃／薰衣草紫小色塊點綴，全程無輪廓。藝術感最強、最成熟，但偏靜態海報感，做成長動畫較吃力。

適合心情小卡、照護海報插圖
