可用性結論

✓
真實外觀「拿得到」：一支乾淨 JSON API（無 CAPTCHA），即時回品名、成分、許可證字號、外觀照片 URL。
✓
官方圖可直連：食藥署 mcp.fda.gov.tw 的圖任何 referer 都能取，最穩定。
!
無結構化外觀欄位：API 不含「形狀/顏色/刻痕」欄位 → 手繪向量無法自動生成（需人工或另接 InfoId=39）。
!
圖散在多家醫院站：部分 http、部分擋熱連（403）→ 不能穩定直連，要快取 → 觸及版權。
!
版權/授權未解：第三方聚合站的圖，正式 App 用之前必須釐清授權。
