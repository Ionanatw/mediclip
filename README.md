# 🌿 mediclip (CareDoc)

> 拍照上傳醫療單，AI 幫你秒懂、秒整理、秒提醒。

醫療照護懶人包 App — 把衛教單、處方箋、回診單變成家人一看就懂的照護指南。

## Status

🚧 開發中 — 先做 SNS 試玩版，再做 App

## Tech Stack

- **Frontend**: Expo (React Native) + Expo Web
- **Backend**: Supabase (Auth + Edge Functions)
- **AI**: Claude API (Sonnet) + OpenAI DALL-E
- **Drug Data**: 食藥署 Open Data API
- **Payment**: RevenueCat
- **Deploy**: Vercel (SNS) + EAS (App)
